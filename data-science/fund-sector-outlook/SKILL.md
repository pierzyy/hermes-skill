# 行业/子类投资展望

## 用户持仓涉及的细分行业
- 🇨🇳 A股：证券、白酒、食品饮料、消费、医疗、信息技术、机器人、红利、宽基指数、金融地产
- 🇭🇰 港股：恒生科技、中国互联网、港股通互联网、港股通红利
- 🇺🇸 美股：纳斯达克、标普500
- 🌍 海外：欧洲、日本、亚太、新兴市场、越南、全球医疗、全球消费
- 🏠 债券：利率债、信用债、纯债、产业债、海外美元债、亚洲债
- 🥇 黄金/商品

## 对每个子行业收集的信息维度

### 1. 行业指数技术面（权重 20%）
- 近1周/1月/3月涨跌幅
- 相对大盘强弱（超额收益）
- 均线位置（5日/20日/60日均线）
- 成交量变化趋势
- 数据源：东方财富行业板块、同花顺行业指数、TradingView

### 2. 行业基本面/估值（权重 25%）
- PE/PB 当前值与历史分位数
- 盈利增速预期（一致预期）
- ROE 趋势
- 股息率
- 数据源：东方财富数据中心、Wind、chioce

### 3. 龙头公司动态（权重 20%）
- 前3大权重股近期公告（业绩预告、重大合同、股东增减持）
- 行业龙头的季报/年报核心数据
- 数据源：巨潮资讯、东方财富个股公告、雪球

### 4. 政策/催化剂（权重 20%）
- 近期/即将出台的产业政策
- 重大事件日历（业绩披露期、行业大会、政策窗口）
- 数据源：国务院/发改委/工信部官网、财联社、华尔街见闻

### 5. 机构研判（权重 15%）
- 最近一周头部券商（中金、中信、华泰、天风等）的行业研报核心观点
- 目标价调整方向
- 数据源：东方财富研报中心、同花顺研报

### 6. 资金面（加分项）
- 北向资金对该行业的配置变化
- 主力资金净流入/流出
- 融资余额变化
- 数据源：东方财富资金流向、Wind

### 7. 情绪面（加分项）
- 雪球/股吧讨论热度（异常升高警惕）
- 百度搜索指数趋势
- 涨停/跌停家数（行业内）

## 评级体系（28维度统一打分）

### 计分规则（全部维度统一量纲）
所有维度计 1-5 分，按权重加权。总权重 = 100%。

### 宏观→维度权重联动 🆕
Step 5 宏观日报输出的 `weight_bias` 参数微调各维度权重：

| 宏观方向 | 技术面(#1) | 估值(#2) | 机构(#5) | 资金(#6) | 期权(#15) |
|---------|-----------|---------|---------|---------|----------|
| 偏进攻 | **+2%** | -2% | +1% | +1% | — |
| 中性 | — | — | — | — | — |
| 偏防御 | -2% | **+2%** | — | -1% | +1% |

进攻市动量更重要（技术面权重提升），防御市安全边际更重要（估值+期权对冲权重提升）。调整幅度控制在 ±2% 以内，不对整体结构造成根本性改变。

### 权重分配（基准，按 weight_bias 微调）

- **核心维度（#1-5）**：1-5 分 × 百分比权重（20%~25%），贡献总分主体
- **加分维度（#6-7）**：1-5 分 × 固定权重 5%，独立于核心
- **加减分维度（#8-13, 20-28）**：1-5 分 × 固定权重 3~10%
- **量化维度（#14-18）**：1-5 分 × 固定权重 3%

所有维度合计权重 = 100%，不再是"加权 + 绝对分"的混合体系。权重分配如下：

| # | 维度 | 权重 |
|---|------|------|
| 1 | 行业指数技术面 | 15% |
| 2 | 基本面/估值 | 15% |
| 3 | 龙头公司动态 | 10% |
| 4 | 政策/催化剂 | 10% |
| 5 | 机构研判 | 8% |
| 6 | 资金面 | 5% |
| 7 | 情绪面 | 5% |
| 8 | 估值极端值检测 | 5% |
| 9-13 | 日历/风格/信用/资金结构/行为金融 | 各 2% |
| 14-18 | 微观结构/期权/QDII/日历/因子 | 各 3% |
| 19 | 盘中实时信号 | 5% |
| 20 | 行业轮动阶段 | 3% |
| 21 | 跨行业资金流向 | 2% |
| 22 | 历史相似期 | 2% |
| 23 | 跨资产相关性 | 3% |
| 24 | 盈利修正动量 | 3% |
| 25 | 产业链传导 | 2% |
| 26 | 行业拥挤度 | 3% |
| 27 | 业绩预期差 | 3% |
| 28 | 行业库存周期 | 2% |

### 非适用维度自动零分
某些维度对特定行业不适用，自动跳过，不参与总分计算：

| 维度 | 不适用行业 | 原因 |
|------|-----------|------|
| 28·库存周期 | 证券/互联网/医疗/消费 | 纯服务业无实体库存 |
| 16·QDII专项 | 纯国内基金组合 | 不涉及跨境 |
| 25·产业链传导 | 债券/货币/黄金 | 无上下游概念 |
| 15·期权信号 | 债市/货基 | 无对应期权市场 |

不适用维度自动标记 N/A，按剩余维度权重等比放大计算总分：
```
总分 = Σ(适用维得分 × 适用维权重) / Σ(适用维权重)
```

### 评级阈值
- 综合分 ≥ 3.8 → 🟢 **增持**
- 3.0 ≤ 综合分 < 3.8 → 🟡 **持有**
- 2.0 ≤ 综合分 < 3.0 → 🔴 **减持**
- 综合分 < 2.0 或数据不足 → ⚪ **观望**

> **校准说明**：实践中 28 维度加权后，≥4.0 几乎无法达成（熊市中即使最被看好的债券也仅 3.85-3.90）。阈值下调至 3.8 使评级分布更合理。熊市/偏防御宏调下债券类可在 3.8-3.9 达到 🟢；牛市/偏进攻下科技/成长类可达 🟢。

## 跨市场联动分析（重要）
- 中美利差 → A股/港股 vs 美股相对吸引力
- 美元指数 → 黄金、QDII基金净值影响
- 美债收益率 → 成长股/科技股估值
- 人民币汇率 → 港股、QDII人民币份额
- 国际油价 → 商品/抗通胀类
- VIX恐慌指数 → 全球风险偏好

## 地缘政治传导分析（重要）
- **中美关系**：关税升级→利好国产替代（信息技术/科创）；缓和→利好出口链（消费电子）
- **中东局势**：霍尔木兹威胁→油价↑→商品基金受益、航空/运输承压
- **俄乌冲突**：能源/粮食供应链→欧洲通胀→ECB政策→欧元汇率
- **台海/南海**：军事紧张→军工板块+黄金避险需求↑
- **供应链**：关键航道中断→全球通胀反复→美联储政策路径改变
- **科技制裁**：芯片出口管制→利好A股半导体（信息技术）但利空港股科技（受限）
- **金融制裁**：SWIFT/美元武器化→人民币国际化↑→利好A股金融
- **选举/政权更迭**：政策不确定性→外资短期流出→汇率波动

### 14. 市场微观结构（量化维度）
- **涨跌家数比**（ADR）：今日A股涨跌比→市场广度
- **新高/新低比**：创52周新高 vs 新低的股票数量→趋势强度
- **量价配合**：指数上涨时成交额是否放大？缩量上涨=弱势反弹
- **涨停/跌停家数**：极端情绪指标（>100家涨停=过热，>50家跌停=恐慌）
- **炸板率**：涨停板被打开的比例→追涨意愿

### 15. 期权/衍生品信号（量化维度）
- **Put/Call Ratio**：>1.2=恐慌底部（反向买入），<0.6=过度乐观
- **VIX期限结构**：期货升水（contango）=正常，贴水（backwardation）=恐慌
- **SKEW指数**：尾部风险定价，高SKEW=市场在定价黑天鹅
- **隐含波动率百分位**：当前IV在历史的什么位置

### 16. QDII专项指标（关键！）
- **ETF溢价率**：QDII ETF 场内价格 vs 实时估算净值(IOPV)，溢价>3%需警惕
- **跟踪误差**：基金净值 vs 基准指数的年化偏离
- **汇率对冲成本**：人民币份额的汇率损耗（CNY vs CNH价差）
- **QDII额度使用率**：基金公司QDII额度是否耗尽（影响申购开放状态）
- **海外节假日**：标的市场休市导致净值暂停更新

### 17. 指数再平衡/事件日历（量化维度）
- **指数调仓日**：沪深300/中证500/科创50等每半年调仓→被动基金跟随
- **港股通标的调整**：调入/调出影响流动性
- **期货/期权到期日**：每月第三个周五→波动放大
- **MSCI/FTSE 季度调整**：外资被动配置变化

### 18. 因子暴露与风险归因（量化维度）
- **当前持仓的因子暴露**：价值/成长/动量/质量/低波/规模
- **因子拥挤度**：哪些因子交易过度拥挤→反转风险
- **最大回撤分析**：各子类从高点的回撤幅度→超跌识别
- **相关性矩阵**：持仓子类之间的相关性→真实分散化程度

### 19. 盘中实时信号 🆕（实时维度，权重 ±1.0）
**适用场景**：14:00 盘中报告。基于实时行情与持仓的交叉判断。

- **子类盘中涨跌幅**：
  - 子类内基金加权平均涨跌幅 > +2% → +0.3（强势）
  - 子类内基金加权平均涨跌幅 < -2% → -0.3（弱势）
  - 区间内 → 0
- **持仓盈亏信号**：
  - 子类盘中浮盈 > +3% → 考虑减仓止盈 → -0.2
  - 子类盘中浮亏 > -3% → 关注超跌机会 → +0.2（逆向）
- **相对强弱（盘中）**：
  - 子类涨幅显著跑赢大盘（超额 >1%）→ +0.2（动量延续）
  - 子类涨幅显著跑输大盘（超额 <-1%）→ -0.2
- **成交量配合**：
  - 涨 + 放量（>20日均量1.5倍）→ +0.2（有效突破）
  - 涨 + 缩量（<20日均量0.7倍）→ -0.1（弱势反弹）
  - 跌 + 放量 → -0.2（恐慌抛售）
  - 跌 + 缩量 → +0.1（抛压衰竭）
- **盘中反转信号**：
  - 开盘跌 >1% 但 14:00 翻红 → +0.2（盘中反转）
  - 开盘涨 >1% 但 14:00 翻绿 → -0.2（冲高回落）
- **持仓集中度风险**：
  - 单子类占总资产 >15% 且当日大跌 → -0.3（集中风险暴露）
  - 单子类占总资产 >15% 且当日大涨 → 不加分（已有收益，不追）

**数据来源**：fundgz API（国内基金实时估值）、ETF 实时行情（QDII 映射）、东方财富板块涨跌

### 20. 行业轮动阶段定位 🆕（周期维度，权重 ±0.5）
基于美林时钟 + 行业轮动理论的阶段判定：

- **四阶段判定**：
  - 📈 **扩张期**：盈利↑ + 情绪↑ + 资金流入 → 顺势持有
  - 🔥 **顶峰期**：估值过热 + 资金开始流出 + 龙头滞涨 → 减仓信号
  - 📉 **收缩期**：盈利↓ + 情绪悲观 + 资金流出 → 观望不动
  - 🌱 **复苏期**：盈利触底 + 情绪修复 + 聪明钱进场 → 加仓窗口
- **阶段指标**：
  - 行业指数 vs 60日均线位置（持续在上方=扩张，持续破位=收缩）
  - 行业ROE趋势（环比连续2季改善=复苏）
  - 分析师盈利预测修正方向（上调占比>下调=扩张）
- **评分**：
  - 扩张期 + 低估值 → +0.5（最佳组合）
  - 扩张期 + 高估值 → -0.1（警惕过热）
  - 收缩期 → -0.3（等信号反转）
  - 复苏期 + 低估值 → +0.5（左侧布局）

### 21. 跨行业资金流向对比 🆕（流量维度，权重 ±0.3）
不是孤立看单个行业，而是看钱在行业间的流动：

- **主力资金净流入/流出排名**（取前5和后5行业）
- **北向资金行业配置变化**（本周 vs 上周，增配/减配方向）
- **ETF 份额变化**（行业ETF份额周环比 → 散户/机构行为）
- **评分**：
  - 本行业连续3日主力净流入 + 排名前5 → +0.3
  - 本行业资金持续流出到其他行业 → -0.3
  - 资金流入但股价不涨（背离）→ -0.2（拉高出货嫌疑）
  - 资金流出但股价不跌（承接强）→ +0.2

### 22. 历史相似期参考 🆕（历史维度，权重 ±0.3）
找到与当前宏观/估值环境最相似的历史时期，参考其后走势：

- **匹配维度**：
  - 当前PE/PB分位数
  - 当前宏观环境（利率方向、通胀水平、PMI趋势）
  - 当前市场情绪（成交量、波动率）
- **参考价值**：
  - 找到 1-2 个相似历史窗口（如"类似2019年Q1"或"类似2022年10月"）
  - 标注该窗口后 1个月/3个月 该行业的实际走势
- **评分**：
  - 历史相似期后续走强 → +0.3（做多参考）
  - 历史相似期后续走弱 → -0.3（警示信号）
  - 找不到显著相似期 → 0（不做强行匹配）

### 23. 跨资产相关性变化 🆕（关联维度，权重 ±0.5）
不是看单一行业，而是看行业与关键资产之间的相关性是否突变：

- **股债相关性**：A股 vs 国债期货，正常为负相关（股涨债跌）
  - 股债同跌 → 流动性危机信号 → -0.5（全面减仓）
  - 股债同涨 → 宽松预期 → +0.3
- **行业与汇率相关性**：人民币升贬对行业影响
  - 航空/造纸受益人民币升值 → 当前汇率方向的传导判断
  - 出口型行业（纺织/家电）反向
- **行业间相关性**：
  - 两个通常不相关的行业突然高度同向 → 系统性风险 → -0.3
  - 相关性正常 → 0

### 24. 盈利修正动量 🆕（领先维度，权重 ±0.3）
比静态 PE/PB 领先 1-3 个月的指标，捕捉分析师预期的边际变化：

- **盈利预测上调/下调比**：最近一个月内分析师上调 vs 下调盈利预测的比例
  - 上调占比 >60% → +0.3（盈利拐点确认）
  - 下调占比 >60% → -0.3（盈利恶化中）
- **连续方向**：
  - 连续 2 周上调占比递增 → +0.3（趋势加速）
  - 连续 2 周下调占比递增 → -0.3（恶化加速）
- **预期差**：
  - 当前行业指数走势 vs 盈利修正方向背离 → 指数可能即将反转
- **数据源**：东方财富盈利预测、Wind 一致预期、chioce分析师预期

### 25. 产业链上下游传导 🆕（联动维度，权重 ±0.3）
A 股板块轮动的核心逻辑：上游涨价挤压下游利润，下游放量拉动上游需求：

- **上下游利润分配**：
  - 上游（煤炭/有色/钢铁）涨价 + 下游（汽车/家电/机械）利润率压缩 → 下游 -0.3
  - 上游降价 + 下游成本改善 → 下游 +0.3
- **需求传导链条**：
  - 地产销售回暖 → 建材/家电/银行 → 相关行业连锁加分
  - 半导体周期上行 → 消费电子/汽车电子/通信设备 → 相关行业连锁加分
- **替代/互补关系**：
  - 光伏景气 → 压缩火电 → 火电 -0.2
  - AI 算力需求↑ → 利好电力/芯片/散热 → 相关行业各 +0.1
- **判断规则**：
  - 本行业处于产业链受益位置 → +0.3
  - 本行业处于产业链受压位置 → -0.3
  - 无明显传导 → 0

### 26. 行业拥挤度 🆕（仓位维度，权重 ±0.5）
机构集中持有的行业容易出现踩踏风险，且与第18维"因子拥挤度"不同——这是持仓层面的拥挤：

- **公募基金超配/低配**：
  - 当前行业在主动基金中的配置比例 vs 基准指数中的权重
  - 超配>2倍基准 → 拥挤（所有基金都买了这个行业 → 增量资金枯竭）→ -0.5
  - 低配<0.5倍基准 → 被忽视（无人关注 → 潜在反转机会）→ +0.3
- **交易拥挤度**：
  - 行业成交额占全市场比例 vs 历史均值
  - >历史2σ → 过度交易 → -0.3
  - <历史-1σ → 冷清 → +0.2（底部特征）
- **分析师覆盖拥挤度**：
  - 覆盖该行业的分析师数量变化趋势
  - 覆盖激增 → 信息已被充分定价 → -0.2
  - 覆盖锐减 → 信息真空 → +0.2（逆向）
- **数据源**：公募基金季报持仓分析、东方财富行业成交额、Wind行业配置

### 27. 业绩预期差 🆕（验证维度，权重 ±0.4）
第24维看的是"上调还是下调"，本维看的是"实际业绩 vs 市场预期的偏差"：

- **业绩超预期/低于预期**：
  - 最近一季报行业中 >60% 个股超预期 → 基本面确认强势 → +0.4
  - 最近一季报行业中 >60% 个股低于预期 → 基本面持续恶化 → -0.4
- **连续趋势**：
  - 连续 2 个季度超预期 → 趋势确认，+0.4（不仅是"惊喜"而是"趋势"）
  - 连续 2 个季度低于预期 → 恶化被低估，-0.4
- **预期修正加速**：
  - 第24维（修正方向）↑ + 本维（超预期）↑ → 双重验证 → +0.5
  - 第24维 ↑ 但本维 ↓（上调但实际低于预期）→ 看空气氛，-0.3
- **数据源**：Wind一致预期、chioice财报分析、上市公司季报

### 28. 行业库存周期 🆕（宏观维度，权重 ±0.4）
库存周期领先盈利数据 1-2 季度，对汽车/电子/化工/服装等周期行业影响巨大：

- **四阶段判定**：
  - 📈 **主动补库存**（需求↑ + 库存↑）→ 繁荣期，行业景气度最高 → +0.4
  - 📉 **被动补库存**（需求↓ + 库存↑）→ 衰退前兆，警惕拐点 → -0.3
  - 📊 **主动去库存**（需求↓ + 库存↓）→ 收缩期，最难阶段 → -0.4
  - 🌱 **被动去库存**（需求↑ + 库存↓）→ 复苏前兆，最佳买点 → +0.4
- **行业匹配**：
  - 电子/半导体：关注芯片库存天数、晶圆厂产能利用率
  - 汽车：经销商库存系数（>2=压力，<1=供不应求）
  - 化工：工业产成品库存同比
  - 服装/纺织：品牌商库存周转天数
  - 纯服务业（如证券/互联网）→ 此维度不适用，计 0 分
- **数据源**：国家统计局工业企业库存、行业协会数据、券商库存周期专题报告

## 自动化评分管线

当需要批量评分 36 个子行业时，应编写 Python 脚本而非逐一手动评分：

1. **输入文件**：`/tmp/fund_report_step4.json`（盘中实时信号，含每个基金的 `gszzl` 估值涨跌）和 `/tmp/macro_daily.json`（宏观研判 + weight_bias）
2. **基金→子行业映射**：基于基金名称关键词手工建立映射表，覆盖全部非货币基金
3. **维度 19（盘中实时信号）**：计算子行业内基金的加权平均涨跌幅 `Σ(amount × gzzl) / Σ(amount)`
4. **债券/海外分化**：债券类因无盘中波动需独立定标（如维度 1 技术面基准 4.5）；海外类需按国别区分（日本 BOJ 紧缩 vs 欧洲 ECB 宽松）
5. **weight_bias 应用**：bearish 宏调下技术面降权 2%、估值升权 2%、期权升权 1%；sector_bias 按行业别微调
6. **NA 维度跳过**：债券跳过 #15(期权)、#16(QDII)、#25(产业链)；证券/医疗/消费跳过 #28(库存周期)
7. **输出格式**：`/tmp/industry_outlook.json`，含 tag（如 `"🟢增持"`）、dimension_scores、na_dimensions、one_liner

### ⚠️ 评分压缩陷阱（2026-05-29 验证）

等权 28 维度自动化评分在平稳市场中存在**严重压缩效应**：所有子行业得分集中在 3.2-3.7 区间，全部落入 🟡持有，无 🟢/🔴 分化。

**原因**：
1. 大部分维度（D3-D18, D20-D28）在无实时数据时取默认值 3.0-3.5
2. 仅 D1（技术面）和 D19（盘中信号）能产生差异化
3. 28 维度 × 0.5-1.0 的范围浮动被等权稀释

**解决方案**（二阶段评分）：
1. **一阶段**：自动化评分产生基线
2. **二阶段**：基于盘中实时信号（D19）和市场极端值进行手动校准：
   - 盘中涨跌幅 >2% 且基本面支持 → 上调 0.3-0.5
   - 盘中跌幅 >2% 且行业承压 → 下调 0.3-0.5
   - 境外无盘中数据子行业按国别/宏观分化手动区分
   - 确保至少产生 🟢≥3、🔴≥3 的评级分布

### ⚠️ 关键词匹配优先级陷阱（2026-05-29）

基金→子行业自动关键词分类时，dict 迭代顺序决定匹配优先级：
- "医疗" 在 dict 中先于 "全球医疗" → 000369 广发全球医疗被误归入 A股_医疗
- "消费" 可能在 "全球消费" 之前匹配 → 118002 易方达标普消费品被误判

**修复**：优先使用**硬编码 code→sub_industry 映射表**，逐代码分配。关键词匹配仅作为兜底。

### 常见陷阱
- **债券得分不足**：熊市中债券是唯一安全资产，维度 1/5/6/10/20 需给到 4.0-4.5 才能让总分过 3.8 阈值
- **海外缺乏分化**：无盘中数据的海外子行业容易全部同分，需在维度 1/2/3 按国别手动区分
- **证券/周期类承压**：熊市+期货贴水环境下证券天然得分最低（2.5-2.9），这符合逻辑不要强行上修
