# 基金组合持仓分析

收到 FundMonitor CSV 导出后，按以下规则分析：

## 分类规则（按基金名称关键词，优先级从高到低）

1. **💰 货币**：名称含 `货币`、`钱袋子`、`招益宝`、`增金宝`、`滚钱宝`
2. **🏦 养老金**：名称含 `养老`
3. **🥇 黄金/商品**：名称含 `黄金` 或 `抗通胀`
4. **🌐 海外债券**：名称含 `美元债`、`亚洲收益债券`、`亚洲美元债`、`亚太精选债券`、`亚洲票息`、`全球美元债`、`全球债券`
5. **🏠 国内债券**：名称含 `纯债`、`国开债`、`信用债`、`利率债`、`产业债`、`双债`、`中债`、`稳泰`、`裕丰`、`双利`、`华嘉`、`债券`
6. **🇭🇰 港股**：名称含 `恒生`、`港股通`、`海外中国互联网`、`海外互联`
7. **🇺🇸 美股**：名称含 `标普500`、`纳斯达克`、`纳指`、`美国成长`、`美国标普`
8. **🌍 海外其他**：名称含 `DAX`，或 `QDII` + `FOF`，或 `全球` + (`QDII` 或 `配置`)，或仅含 `QDII`
9. **🇨🇳 A股权益**：名称含 `指数`、`混合`、`股票`、`增强`、`ETF联接`、`红利`、`消费`、`医疗`、`医药`、`白酒`、`食品`、`证券`、`机器人`、`信息技术`、`金融地产`、`A50`、`A500`、`沪深300`、`中证500`、`国证2000`、`科创`

## 聚合规则

- 不同组合内的**相同基金**（同代码）合并为一只，金额相加
- 不展示组合名称
- 每个分类下所有基金必须全部列出，不可省略

## 输出格式

1. 先展示总金额和去重后基金数量
2. 汇总表格：大类名、金额、占比，按金额降序排列
3. **每个大类下紧跟完整基金列表**（代码、名称、金额），与汇总一起输出，不要分开
4. ASCII 柱状图一览
5. **金额不要千分位逗号**（如 ¥298824.70 而非 ¥298,824.70）
6. 相同基金跨组合合并总额
7. **基金列表每行一只，用 `- 代码 名称 ¥金额` 格式**，确保分行清晰

## 占比分类（9大类）

🇨🇳 A股权益 / 💰 货币 / 🇭🇰 港股 / 🏠 国内债券 / 🇺🇸 美股 / 🌍 海外其他 / 🏦 养老金 / 🌐 海外债券 / 🥇 黄金/商品

## 大类内细分规则

以下6大类需要细分，💰货币、🏦养老金、🥇黄金/商品不细分。

### 🇨🇳 A股权益 细分
- **信息技术**：信息技术、信创
- **机器人**：机器人
- **科创**：科创
- **证券**：证券
- **红利**：红利低波、红利质量、标普红利、红利（不含港股通）
- **食品饮料**：食品饮料
- **宽基指数**：中证500、沪深300、A500、A50、国证2000
- **白酒**：白酒
- **医疗**：医疗、医药、健康
- **消费**：消费主题、主要消费、消费行业、消费股票
- **金融地产**：金融地产
- 剩余混合型归入对应主题或「主动混合」

### 🇭🇰 港股 细分
- **恒生科技**：恒生科技
- **中国互联网**：海外中国互联网、海外互联
- **港股通互联网**：港股通互联网
- **港股通红利**：港股通高股息、港股通红利
- **恒生ETF**：恒生（非科技/互联/红利类）

### 🏠 国内债券 细分
- **利率债**：国开债、国开行、利率债
- **纯债**：纯债、稳泰、裕丰、双利、华嘉、丰享、招悦、及其他未命中关键词的债券
- **信用债**：信用债
- **产业债**：产业债
- **双债/可转债**：双债

### 🇺🇸 美股 细分
- **纳斯达克**：纳斯达克、纳指（优先于标普500匹配）
- **标普500**：标普500、标普100、等权重（不含纳斯达克关键词）
- **美国成长**：美国成长

### 🌍 海外其他 细分
- **全球配置**：全球多元、全球配置、全球精选、全球策略
- **全球医疗**：全球医疗
- **欧洲**：欧洲、德国、DAX
- **日本**：日本
- **新兴市场**：新兴市场
- **亚洲**：亚洲机会
- **亚太**：亚太
- **全球消费**：消费品
- **越南**：越南
- 其余QDII归入「全球股票」

### 🌐 海外债券 细分
- **亚洲债**：亚洲收益债券、亚洲美元债、亚太精选债券、亚洲票息（优先匹配，避免被美元债关键字误匹配）
- **美元债**：美元债、全球美元债（不含亚洲美元债）
- **全球债券**：全球债券
