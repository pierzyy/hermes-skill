# 场外纳指/标普 QDII 基金申购状态监控 v2.1

## 触发命令

用户说「**QDII监控**」或「**QDII申购状态**」即触发。直接运行脚本即可。

## 快速执行

```bash
python3 /opt/data/scripts/qdii_monitor.py
```

## 核心流程

```
fundcode_search.js (2.7万只基金) → 筛选QDII/海外指数 ~733只
  → 名称分类 + 排除场内ETF → 分类查限额
  → 过滤限额≥100元 → 对比昨日缓存 → 输出
```

## 分类规则

| 类别 | 标识 | 判断逻辑 |
|------|------|----------|
| **纳斯达克** | `#纳斯达克#` | 名称含"纳斯达克/纳指" |
| **标普500** | `#标普500#` | 名称匹配"标普500"或"S&P 500" |
| **标普行业** | `#标普行业#` | 名称含"标普"但非追踪标普500指数（如消费品、油气、医疗） |
| **候选** | `#候选#` | 名称含"美国成长/科技/精选/品质"的主动型基金，可能布局纳指标普成分股 |

## 排除规则

- **场内ETF**：代码以 159/513/518/516/562/563/517/588 开头 → 排除，无申购限额
- **场内LOF**（16xxxx）：保留，有场外申购通道
- **纯债/商品/REITs**：由其名称自动分不到任何类别，自然排除

## 输出格式（紧凑，减少token）

```
#纳斯达克#
017894|汇添富纳斯达克生物科技ETF联接A|5,000
...
539001|建信纳斯达克100指数(QDII)A人民币|100 ↑
#标普500#
050025|博时标普500ETF联接A|100
...
#标普行业#
162415|华宝标普美国消费人民币A|1,000
...
#候选#
000043|嘉实美国成长股票人民币|100
```

- 三列：代码 | 名称 | 限购额
- `↑` 标记：限额相比上次查询有变化
- 无装饰字符、无进度输出

## Token 优化措施

| 措施 | 效果 |
|------|------|
| 基金列表缓存7天 | 避免每次下载2.2MB的 fundcode_search.js |
| 结果缓存 + 变化标记 | 一眼看出哪些限额变了，不需要逐行对比 |
| 紧凑输出格式 | 无表格边框、无进度条、纯数据管道 |
| 排除场内ETF | 减少不必要的查询（~40只ETF免查） |
| 15线程并行 | 150只≈55秒，减少agent等待时间 |

## 脚本路径

`/opt/data/scripts/qdii_monitor.py`

## 注意事项

1. fundmobapi 已被反爬封禁（返回"网络繁忙"），不能依赖API
2. api.fund.eastmoney.com 有路由限制，也不能直接调
3. 最稳定方式：直接爬 fund.eastmoney.com/{code}.html 页面
4. 限额信息在HTML中明文存在，不需要浏览器渲染
5. 多线程15路并行是安全值，实测不触发反爬
